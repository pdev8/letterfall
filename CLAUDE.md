# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

The repo guide — commands, the PR-only workflow, architecture, the game-term→code
vocabulary, rule invariants, and UI conventions — lives in AGENTS.md (shared with
all coding agents) and is imported below. Before nontrivial work, also read
docs/ROADMAP.md (canonical plan: epics, LF-xxx tickets, scoring/difficulty/
leaderboard/achievement specs) and docs/GENERATION.md (the Living Deck — the
core game design). When the roadmap changes, docs/roadmap.html changes with it
and gets republished as a Claude artifact in the same PR.
